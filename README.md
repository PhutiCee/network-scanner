## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

## Install the package:

```bash
pip install -e .
```

## Install with development dependencies:

```bash
pip install -e ".[dev]"
```

#Running Tests
This project uses pytest.

Run the test suite:

```bash
pytest tests/ -v
```
